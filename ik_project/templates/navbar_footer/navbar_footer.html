About me

My name is Pim Hakkert and I'm a student in the MBO 4 course of Game- / Media-development. I aspire to be a full time web developer by the end of this course. This website was created as my first project in my first year of the course. Every single line of HTML, CSS, JavaScript and PHP is written by hand to showcase my skills. The contents of this website go beyond front end development. We had to include a Content Management System, so I decided to start from zero and I learned basic PHP to help me create a working CMS. My hopes are that by viewing this website, you are interested in hiring me for your future projects. To see a good example of my skillset, please visit the "portfolio" page.
